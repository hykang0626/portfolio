Jikko UI 컴포넌트 가이드 데모

design.md에 부합하는 라이트 테마, 굵은 테두리(Bold Outline) 컴포넌트들을 한눈에 확인하는 페이지입니다.

1. 내비게이션 바 (Navbar)

2. 자기소개 프로필 (Profile)

3. 프로젝트 카드 (ProjectCard)

마우스를 올렸을 때(Hover) 네온 대신 굵은 검은색 하드 섀도우가 우측 하단으로 더 돌출되는 인터랙션을 지원합니다.

4. 이메일 문의 폼 (ContactForm)

5. 상세 보기 모달 팝업 (Modal)

아래 버튼을 누르면 데이터가 주입된 둥근 모달 상세 보기 팝업이 화면에 출력됩니다. (유튜브 영상 테스트 포함)
